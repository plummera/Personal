PROJECT GHOSTNET

 

Project GhostNET is a collection of computers, connected to the internet, that interact with specific nodes to accomplish distributed tasks. GhostNET typically connects networked computers to each other allowing them to communicate and function as part of a bigger, thus more capable network for larger tasks. Such systems are composed of compromised machines that are assimilated through instruction, interrupt, or malfunction, but generally through user permission for GhostNET activity.
